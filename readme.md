# Log Extraction Challenge

## Overview
Efficient log extraction from a large (1 TB) log file using Python's memory mapping technique.

## Setup
1. Clone the repository
2. Download the log file
3. Run the extraction script

## Requirements
- Python 3.7+
- `test_logs.log` file in the same directory

## Usage
```bash
python extract_logs.py YYYY-MM-DD
```

## Output
Extracted logs will be saved in `output/output_YYYY-MM-DD.txt`
